• git init :

    Initialize a new Git repository

  • git clone :

    To clone a remote repository to your machine

  • git status :

    Shows the cuurent status of the working directory

  • git add :

    Add file(s) to the staging area(working area)

  • git commit -m 'message' :

    Save changes to the repo

  • git push :

    Upload commits to the remote repository

  • git pull :

    Fetch and merge changes from the remote repo

  • git branch :

    List or create branches

  • git checkout :

    To switch to another branch i.e., from main to any/ any branch to main

  • git merge :

    To merge another branch into the current one

  • git reset --hard (code of previous version) :

    Resets the whole application and code structure to the previous version

    • Before hard reset :

    After hard reset

  • git reset --soft (code of previous version) :

    Resets the application to the previous version but the code remains same

    • As discussed there won't be any change for the code in soft reset
