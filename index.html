Introduction to Git & GitHub

What is Git?

Git is a free and open-source distributed version control system. It helps developers track changes in their code and collaborate with others. It was created by Linus Torvalds in 2005.

What is GitHub?

GitHub is a cloud-based platform that allows you to store, manage, and share your Git repositories. It adds collaboration features like pull requests, issue tracking, and more.

Why are Git and GitHub Important?

  • Track code changes over time
  • Collaborate with other developers
  • Work on multiple features using branches
  • Revert back to previous versions if needed
  • Share your code and projects with the world

Basic Git Commands

  • git init - Initialize a new Git repository
  • git clone <url> - Copy a remote repo to your machine
  • git status - Show the current status of the working directory
  • git add <file> - Add file(s) to the staging area
  • git commit -m "message" - Save your changes to the repo
  • git push - Upload commits to the remote repository
  • git pull - Fetch and merge changes from the remote repo
  • git branch - List or create branches
  • git checkout <branch> - Switch to another branch
  • git merge <branch> - Merge another branch into the current one

Conclusion

Git and GitHub are essential tools for modern software development. Whether you're working solo or with a team, learning how to use them will greatly enhance your productivity and project organization.
